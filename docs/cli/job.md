# Add
Add a new job definition

Usage: 
<code>dotnet pc.dll job add --project [project] --name [name]</code>

**Options**
<details>
    <summary>project (mandatory)</summary>
    <p>
        <code>--project</code> (alias: <code>-p</code>)
    </p>
    <p>
        Name of the project
    </p>
</details>
<details>
    <summary>name (mandatory)</summary>
    <p>
        <code>--name</code> (alias: <code>-n</code>)
    </p>
    <p>
        Name of the new job definition
    </p>
</details>

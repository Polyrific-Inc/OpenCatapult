# Remove
## Overview
Remove a user

Usage: 
<code>dotnet pc.dll account remove --email [email]</code>

## Options
<details>
    <summary>email (mandatory)</summary>
    <p>
        <code>--email</code> (alias: <code>-e</code>)
    </p>
    <p>
        Email of the user to be removed
    </p>
</details>
